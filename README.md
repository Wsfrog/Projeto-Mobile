# Projeto Java Web com JSP

Este é um projeto simples desenvolvido com Java Web utilizando JSP, Servlets e JDBC. O objetivo é demonstrar o funcionamento básico de um sistema web dinâmico.

## 🛠 Tecnologias Utilizadas

- Java 8+
- JSP (JavaServer Pages)
- Servlets
- JDBC
- Apache Tomcat
- MySQL
- HTML/CSS/Bootstrap (opcional)
